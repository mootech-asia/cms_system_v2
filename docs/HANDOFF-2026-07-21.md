## 業主目標(本 session 再次確認,最重要)

**整個工廠(v2 前台 + `/studio` 設計後台)與交付版(v3)全部要變成純 HTML+CSS+JS,
而且是「手寫、人類看得懂」的乾淨切版** —— 這是設計師交付物(切版 + 完整頁面功能)。
工程師拿這份完整切版,再自己去長他要的框架版本(那是工程師的下游步驟,不是工廠的事)。

**目前 bug 的根因:** 當初 Nuxt→靜態 / Vue→靜態 的轉換,是把「編譯產物」倒出來 +
只做結構驗證,沒有做「功能完整對齊原版 + 人類可讀」。所以功能被漏搬(收藏、語系、
頁籤等)、程式碼是壓縮/編譯的(如 `entry.Bu_SodWY.css` 一行 104KB)。

**新標準(業主鐵則):完成 = 功能完整對齊原版 + 程式碼人類可讀 + 實際在瀏覽器操作驗過**
(不是結構綠燈就算)。
